This app helps users to read Welsh language articles, without needing to translate an entire article. The ambition is to create a widget that can be easily installed (one line of code) on any HTML page.

Geurfa uses 'Eurfa' data from Kaggle: https://www.kaggle.com/datasets/rtatman/eurfa-welsh-dictionary

Anyone is welcome to modify, re-use or improve Geurfa in any way and without permission.

Restrictions:
- The app does not recognize words that have been mutated and some other forms
- There must be an HTML <header> element on a website in order to install the widget
- The Geurfa database is a bit limited compared to others
